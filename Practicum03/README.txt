encoder, Randy Thiemann 2013
----------------------------

Dit programma is getest op OS X en Windows en geeft op beide systemen de output
files die te vinden zijn in de bin map. In de bin map vindt u ook een windows
executable gebuild met VS2012. Gelieve voor OS X en linux de source zelf te
builden, dit wegens grote variatie in libraries in verschillende distros.

Op Windows geeft cmake een werkende VS2012 solution file. Op OS X maakt u best
een map build aan, en in die map typt u dan "cmake .. && make". Dit geeft een
werkende executable.
